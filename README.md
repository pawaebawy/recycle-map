# ЭкоЧек — MVP прототип

Прототип для проверки гипотезы: пользователи могут отмечать мусор, брать задачи на уборку и получать за это баллы.

## Стек

- React + Vite
- Zustand (состояние)
- OpenLayers + OpenStreetMap
- Tailwind CSS

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173/
Для сборки: `npm run build`

## Функционал

1. **Регистрация/вход** — email и пароль (данные в localStorage)
2. **Карта** — OpenLayers с кластеризацией (жёлтые — активные, зелёные — выполненные)
3. **Страница задачи** — `/task/:id` с полной информацией
4. **Создание задачи** — метка на карте, фото, заголовок, описание
5. **Взятие задачи** — кнопка «Взять» на карточке
6. **Выполнение** — загрузка фото «после», проверка геолокации (радиус 50 м)
7. **Проверка** — проверяющий подтверждает или отклоняет
8. **Баллы** — 10 за задачу, рейтинг участников
9. **Профиль** — статистика и список задач

## Данные

- Начальные данные: `public/data.json`
- Сохранение: `localStorage` (ключ `app-data`)
- Сброс: удалить `app-data` в DevTools → Application → Local Storage
